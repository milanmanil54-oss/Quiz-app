## Quiz App (HTML, CSS & JavaScript)
A simple interactive Quiz Application built using HTML, CSS, and JavaScript.
This project demonstrates DOM manipulation, event handling, score tracking, and dynamic question rendering.

## Features
Multiple-choice quiz questions
Score calculation system
Instant feedback after answering
Responsive and clean UI
Restart quiz option

## Technologies Used
HTML5 – Structure
CSS3 – Styling & layout
JavaScript (Vanilla JS) – Logic and interactivity
